Name Assignment

Description

Use scratch to design and create a program that shows my name. The requirements were to add a sprite for each letter in my name and give them 3 unique behaviors. The program needed to run the green flag was clicked and for extra credit I added a function to reset that scene to the beginning each time the green flag was clicked.

Concepts Learned:

  • Using event blocks , green flag clicked, when key pressed
  • Conditional statements, used if statements to do a motion if statement is true
  • Used loops, I used a forever block to continue motion forever
  • I used a repeat until block to repeat a motion a certain number of times
  • I used the library of sprites to find letter in my name and adjusted the costumes
  • Movement, I used the x and y values to make my sprites move to their original location when the green flag was clicked, I rotated the sprite by using motions
